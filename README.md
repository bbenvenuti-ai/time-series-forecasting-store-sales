# time-series-forecasting-store-sales

## Description
This is a complete time series forecasting system that forecasts a business metric and quantifies uncertainty.

## Dataset

[Store Sales - Time Series Forecasting](https://www.kaggle.com/competitions/store-sales-time-series-forecasting/data)
Originally obtained via this command:
```
 kaggle competitions download -c store-sales-time-series-forecasting
 ```
Why this dataset?
- Clean but realistic
- Contains multiple time series
- Known evaluation pitfalls
- Excellent for feature engineering
